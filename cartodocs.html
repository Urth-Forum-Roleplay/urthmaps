Map Measurements

The easiest way to measure area is to use the Fuzzy Select tool in GIMP in combination with the Histogram window (Windows>Dockable Dialogues>Histogram). The window displays a pixel count at the bottom, which can then be multiplied by one of the area values below.

  • An individual pixel on the map is exactly 6.34126 km2 (2.44837 mi2)
  • An individual pixel on the map is exactly 2.518 km (1.565 mi) tall/wide
