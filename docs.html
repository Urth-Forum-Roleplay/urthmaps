Map Measurements

The easiest way to measure area is to use the Fuzzy Select tool in GIMP in combination with the Histogram window (Windows>Dockable Dialogues>Histogram). The window displays the number of selected pixels at the bottom, which can then be multiplied by one of the area values below. This process should be done on the political map XCF file and not a PNG.

  • A pixel on the map is equivalent to 6.34126 km² (exactly 50 ÷ 2.808²)
  • The side of a pixel on the map is 2.518 km long (exactly √50 ÷ 2.808)

Procedure for Making the "HRUM_without_names" Map

  1. Hide legend, update info, all labels
  2. Duplicate Main Political Layer and verify that the alpha-channel is locked
  3. Color-select the black borders, go to Select > Grow in the top bar and grow the border by 1 pixel
  4. Set the Fill tool to Fill whole selection and fill the entire selection with black
  5. Unlock alpha channel, Ctrl-I to invert the selection, and delete the part of the layer outside the border
  6. Set opacity of this border layer to 40%
  7. Recolor the original Main Political Layer countries and disputed territories and the Ocean layer according to the colors on the current HRUM_without_names.png
  8. Merge all visible layers together
  9. Use the scale tool, with interpolation set to Cubic, to scale the image to 4269x2860
  10. Export
